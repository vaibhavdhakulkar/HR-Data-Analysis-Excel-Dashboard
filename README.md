Project Title: HR Analytics Dashboard (Excel)
This project presents an interactive Excel Dashboard designed to analyze employee data across departments and work locations.

Key Features:
Visualized employee count by employment status (Full-Time, Part-Time, Contract)
Salary and leave comparison by department and role
Age and gender distribution insights
Regional and workplace analysis
Department-wise average performance ratings
Overview of employee skills, certifications, and training programs

Skills & Tools: Excel, Pivot Tables, Charts, Data Cleaning, Dashboard Design, HR Analytics
